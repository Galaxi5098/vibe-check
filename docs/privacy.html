Vibe Check — Privacy Policy

Last updated: April 13, 2026

Overview

Vibe Check is a Chrome extension that performs lightweight sentiment analysis on webpage text and displays a mood emoji in the toolbar. Your privacy is important, and this extension is designed to collect as little data as possible.

Data Collection

This extension does not collect, store, transmit, or share any personal data. It does not use analytics, tracking, cookies, or any form of telemetry.

Website Content

The extension reads visible page text to perform sentiment analysis using a local word list. All processing happens entirely in your browser. No page content, URLs, or analysis results are sent to any external server.

Local Storage

The extension stores your enabled/disabled preference locally using Chrome's built-in storage API. This data never leaves your device.

Permissions

  • Host permissions (all URLs) — To read page text for sentiment analysis on every webpage.
  • storage — To save your enabled preference locally.

Third Parties

This extension does not share data with any third parties. There are no ads, analytics, or external services. No API calls are made.
